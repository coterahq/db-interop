# dbt-interop

To install dependencies:

```bash
bun install
```

To bundle:

```bash
bun run build
```


To run:

```bash
bun run __tests__/run.ts
```

This project was created using `bun init` in bun v1.0.26. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
